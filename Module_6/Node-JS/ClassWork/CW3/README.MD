## 💻 Installation & Usage

### ➤ Install using npm

```bash
npm install binarysearch_geek
````

### ➤ Example

```js
const binarySearch = require('binarysearch_geek');

const arr = [1, 3, 5, 7, 9];
const target = 7;

console.log(binarySearch(arr, target));  // Output: 3
```
